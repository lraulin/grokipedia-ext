Grokipedia Extension Test Page

Note: When you click these links with the Grokipedia extension installed, they should redirect to grok.com instead of wikipedia.org

Test Links

Click any of these Wikipedia links to test the redirect:

Expected Behavior

  • https://en.wikipedia.org/wiki/Main_Page → https://grok.com/en/wiki/Main_Page
  • https://es.wikipedia.org/wiki/Wikipedia:Portada → https://grok.com/es/wiki/Wikipedia:Portada
  • All language variants should be preserved
  • All path components should be preserved

Installation Instructions

  1. Open Chrome and navigate to chrome://extensions/
  2. Enable "Developer mode" (toggle in top-right)
  3. Click "Load unpacked"
  4. Select the grokipedia-ext directory
  5. Return to this page and test the links above
